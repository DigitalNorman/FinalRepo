Sustainability

The site is a static HTML site which is hosted on GitHub Pages. All the pages are plain HTML and the database’s primary files are saved as a CSV file. However, the map will require Python and the 'Folium' library to be viewed. All the pages are linked to a single style sheet. All the files in the repository are those that create the site. The file titled 'data_cleaning_methodology.md' in the repository contains the details of my data cleaning process.

Dependencies

Python and the 'Folium' library for creating interactive maps is required in order to view the map. More instructions can be found in the 'README' in the repository.

Maintenance

To update or continue the project, one would need:

  • A GitHub account and access to the repository.
  • A web browser to view the changes.
  • Basic knowledge of HTML, CSS, and JavaScript.
  • Python and the 'Folium' library for creating interactive maps.
