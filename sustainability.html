Sustainability

The site is a static HTML site which is hosted on GitHub Pages. All the pages are plain HTML and the database’s primary files are saved as a CSV file. However, the top most table is a Java script (map has not been included yet here). All the pages are linked to a single style sheet. All the files in the repository are those that create the site.

Dependencies

There is currently a single JavaScript used to create the primary table. Your machine will have to be able to read in the language in order to view the table.

Maintenance

To update or continue the project, one would need:

  • A gitHub account and access to the repository.
  • A web browser to view the changes.
  • Basic knowledge of HTML, CSS, and JavaScript.
  • JavaScript runtime environment.

Currently no other tools are needed (this may change soon if the map is added).
